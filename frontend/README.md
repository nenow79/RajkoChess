# Chess App Frontend

Frontend aplikacji szachowej oparty o React i Vite.

Najczęściej uruchamiaj projekt z katalogu głównego przez:

```bash
./start.sh
```

Uruchomienie samego frontendu:

```bash
npm install
npm run dev
```

Szczegóły konfiguracji backendu i pełny opis projektu są w głównym `README.md`.
